Detailed Description

Represents an employee in the company.

Employee class is an abstract class that contains information like name, salary, and ID badge of the employee. It hasvirtual methods for employee tasks, attending meetings, and receiving salaries, which are to be overridden by derived classes Manager and Developer. The class also includes a method to set the performance percentage of the employee which is used in manager class for perfomance review.
